1. 인코딩 유형 에러 
UTF-8 서명 없음으로 인코딩 되어있는 파일들 전부다 UTF-8 DOM 서명 있음으로 인코딩 해주기
Visual Studio 프로젝트 루트 폴더에서 PowerShell 실행

방법: 경로에서 Shift + 마우스 오른쪽 → PowerShell 창 열기

아래 명령어 실행:

powershell
복사
편집
Get-ChildItem -Recurse -Include *.cpp,*.h,*.c,*.hpp | ForEach-Object {
    $content = Get-Content $_.FullName -Raw
    $utf8bom = [System.Text.Encoding]::UTF8
    [System.IO.File]::WriteAllText($_.FullName, $content, $utf8bom)
}
